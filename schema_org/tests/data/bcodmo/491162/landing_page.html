Description

Nitrogen fixation rates derived from acetylene reduction assays

Acquisition Description

Sampling and Analytical Methodology:
Samples were drawn from CTD mounted Niskin bottles. N2 fixation measurements were
carried out using the acetylene (C2H2) reduction method described by Capone [1993].

Related files and references:
Capone, D. G. (1993), Determination of nitrogenase activity in aquatic samples using the acetylene reduction procedure, in Handbook of Methods in Aquatic Microbial Ecology, edited by P. F. Kemp et al., pp. 621–631, Lewis Press, Boca Raton, Fla

Processing Description

Data Processing:
Sample analysis in progress, data not yet available

CTD Sea-Bird SBE 911plus
(more info)
Short Name: CTD SBE 911plus
Community Standard Description
Generic Description

The Sea-Bird SBE 911plus is a type of CTD instrument package for continuous measurement of conductivity, temperature and pressure. The SBE 911plus includes the SBE 9plus Underwater Unit and the SBE 11plus Deck Unit (for real-time readout using conductive wire) for deployment from a vessel. The combination of the SBE 9plus and SBE 11plus is called a SBE 911plus. The SBE 9plus uses Sea-Bird's standard modular temperature and conductivity sensors (SBE 3plus and SBE 4). The SBE 9plus CTD can be configured with up to eight auxiliary sensors to measure other parameters including dissolved oxygen, pH, turbidity, fluorescence, light (PAR), light transmission, etc.). more information from Sea-Bird Electronics

PI supplied instrument name: CTD SBE 911plus
Dataset-specific description
Niskin bottle
(more info)
Short Name: Niskin bottle
Community Standard Description
Generic Description

A Niskin bottle (a next generation water sampler based on the Nansen bottle) is a cylindrical, non-metallic water collection device with stoppers at both ends. The bottles can be attached individually on a hydrowire or deployed in 12, 24 or 36 bottle Rosette systems mounted on a frame and combined with a CTD. Niskin bottles are used to collect discrete water samples for a range of measurements including pigments, nutrients, plankton, etc.

PI supplied instrument name: Niskin bottle
